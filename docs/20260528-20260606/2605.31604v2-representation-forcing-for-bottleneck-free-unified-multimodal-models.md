## 摘要
统一多模态模型旨在用单个模型处理感知和生成任务。然而，现有的统一多模态模型仍依赖于冻结的、单独预训练的变分自编码器进行图像生成，从而带来了结构瓶颈。简单移除变分自编码器会导致质量差距，因为模型必须从原始像素中同时学习高层结构和低层细节。本文提出表示强制技术，通过使表示预测成为模型的原生能力来消除这一差距。具体来说，表示强制强制解码器将视觉表示作为中间标记在像素之前自回归地预测；这些标记随后保留在上下文中，以指导同一主干网络内的像素扩散。通过将表示从感知输出转变为生成目标，表示强制消除了对外部生成潜在空间的需求。我们发现表示强制对理解和生成任务均有裨益。在图像生成方面，采用表示强制和像素空间的模型与基于变分自编码器的最先进统一模型性能相当。在图像理解方面，像素空间表示强制通常优于其基于变分自编码器的变体。这些结果共同为迈向端到端、无瓶颈的统一多模态模型提供了有效的一步。

## Abstract
Unified multimodal models (UMMs) aim to handle perception and generation in a single model. Yet existing UMMs still rely on a frozen, separately pretrained VAE for image generation, imposing a structural bottleneck. Naively removing it introduces a quality gap, as the model must learn both high-level structure and low-level details from raw pixels. In this paper, we propose Representation Forcing (RF), a technique that closes this gap by making representation prediction a native capability of the model. Concretely, RF forces the decoder to autoregressively predict visual representations as intermediate tokens before pixels; these tokens then stay in context to guide pixel diffusion within the same backbone. By turning representations from perception outputs into generation targets, RF eliminates the need for any external generative latent space. We find that RF benefits both understanding and generation. On image generation, our pixel-space model with RF matches state-of-the-art VAE-based unified models. On image understanding, pixel-space RF generally outperforms its VAE-based variant. Together, these results offer an effective step toward end-to-end, bottleneck-free UMMs.

- **现有统一多模态模型的结构瓶颈**：当前统一多模态模型（UMMs）在处理图像生成时，普遍依赖一个单独预训练并冻结的 VAE（变分自编码器）。VAE 将像素压缩到潜空间，再用固定解码器恢复。这个潜空间是针对重建优化而非统一模型的目标，且其有损压缩为生成质量设定了硬上限，无法通过后续训练突破。
- **直接移除 VAE 的困难**：如果在统一模型中直接去除 VAE，在像素空间中生成图像，模型必须同时学习高层语义结构和低层细节，导致生成质量明显下降。
- **本文目标**：提出一种方法消除 VAE 瓶颈，使统一模型能够在像素空间中实现高质量生成，同时不牺牲甚至提升理解能力。最终目标是实现端到端、无外部预训练组件的统一多模态模型。

## 2. 方法论：核心思想、关键技术细节

### 核心思想：表示强制（Representation Forcing, RF）
- 让解码器在生成像素之前，先自回归地预测视觉表示（representation tokens）作为中间 token。这些预测出来的表示 token 留在序列中，通过共享的自注意力机制为后续的像素扩散提供结构性引导。
- 视觉表示来源于模型自身的理解编码器，通过在线向量量化（online VQ）将其离散化为 token 序列。这样，理解和生成共享同一个表示空间，无需外部潜空间。

### 关键技术细节
1. **表示提取与离散化**：
   - 使用理解编码器（DINOv3 ViT-H+/16）的 EMA 副本提取 patch 级特征。
   - 通过在线向量量化（码本大小 K=16384，动量更新 + Sinkhorn-Knopp 平衡）将每个 patch 特征映射为离散 token 索引，形成与空间布局对应的 token 序列。
2. **解码器训练**：
   - 文本 token（T）和表示 token（R）使用因果注意力，进行自回归预测（交叉熵损失 \(L_{LM}\) 和 \(L_{Rep}\)）。
   - 像素 patch（P）采用双向注意力，但只向前关注文本和表示 token（因果边界）。像素生成使用流匹配（flow matching），预测干净像素 \(x_\theta\)，损失为速度损失 \(L_{FM}\)。
3. **训练与推理流程**：
   - 训练时：输入文本和真实图像，通过 EMA 编码器提取表示 token 作为教师强制目标，解码器学习预测表示 token 并同时进行像素扩散。
   - 推理时：编码器不参与。解码器先从文本自回归生成表示 token 序列，然后以这些表示 token 和文本为条件，从高斯噪声迭代去噪得到像素图像。
4. **架构**：采用 Mixture-of-Transformers（MoT）架构，所有 token 共享自注意力层，但路由到不同的前馈专家（理解、表示预测、像素生成）。初始化自 Qwen3-A3B 语言模型。

## 6. 主要结论与发现

1. **RF 消除了 VAE 瓶颈**：像素空间模型使用 RF 后，图像生成质量（GenEval 0.84，DPG-Bench 84.15）匹配或优于基于 VAE 的统一模型（如 BAGEL 0.82，84.07），且带有丰富纹理细节。
2. **RF 同时提升理解与生成**：在理解 benchmark 上，Pixel+RF 优于 VAE+RF 在 6/8 个指标上，表明像素空间生成与统一多模态建模更兼容。
3. **RF 对像素空间生成至关重要**：无 RF 时像素空间 GenEval 仅 0.25，启用 RF 后跃升至 0.76（匹配 VAE 的 0.77）。
4. **离散表示预测优于连续回归**：离散 token（0.76）远优于连续特征预测（0.26），离散化促进了结构细节的分离。
5. **RF 优于辅助对齐（REPA）**：直接预测表示 token 作为条件（0.76）比隐式特征对齐（0.43）有效得多。
6. **理解编码器选择 DINOv3 优于 SigLIP2**，因为自监督特征具有更丰富的空间结构。

## 7. 优点

- **方法简洁有效**：RF 通过将表示预测作为自回归目标，自然地融入现有统一模型框架，无需额外复杂的模块或训练技巧。
- **消除外部依赖**：完全移除预训练 VAE，实现了真正的端到端统一模型，理解和生成共享同一表示空间。
- **实验控制严谨**：通过多组控制对比和消融，清晰揭示了每个设计选择的影响。
- **双重提升**：RF 不仅挽救像素空间生成质量，还意外地提升了理解性能，表明表示共享对两者都有益。
- **可扩展性**：方法可推广到其他基于序列的统一模型架构。

## 8. 不足与局限

- **初始化依赖预训练 LLM**：模型从 Qwen3-A3B 初始化，而非从随机初始化进行完全的多模态端到端训练，可能尚未充分探索联合表示空间。
- **仅限静态图像**：未将 RF 扩展到视频或其他时序模态，限制了应用范围。
- **理解评估在预训练阶段**：未使用后训练（如 RLHF）等优化，因此理解结果可能不是最优状态，与经过充分调优的模型直接比较可能不完全公平。
- **文档型任务有小幅下降**：在 DocVQA 和 ChartQA 上，Pixel+RF 相比无 RF 有轻微下降（-2.0 和 -0.4），表明 RF 的表示结构可能不利于精确文本识别和布局解析。
- **计算开销**：RF 引入了表示 token 的自回归预测，在推理时增加了一个额外的生成阶段，但论文未量化额外成本。
- **未讨论潜在风险**：虽然附录提到标准安全措施，但未深入分析 RF 可能引入的特殊安全或偏见问题（如表示 token 的偏见传播）。
