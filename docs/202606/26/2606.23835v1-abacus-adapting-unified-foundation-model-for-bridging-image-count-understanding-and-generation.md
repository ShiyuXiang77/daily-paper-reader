## 摘要
ABACUS是一个统一的视觉语言模型，可以处理目标计数、人群计数、指代表达式计数以及计数忠实图像生成，无需任何基准特定训练。我们的模型建立在现有的3B参数统一基础模型上，并针对目标定位任务进行了适配，采用了三个关键创新：利用目标性图进行密度感知自适应缩放以实现空间定位；通过GRPO实现边界感知计数策略以消除裁剪边界误差；以及一种循环一致性GRPO策略，其中理解分支对生成输出进行自我批评，从而在无需任何外部标注的情况下缩小理解与生成之间的差距。ABACUS在七个基准测试中取得了最先进的结果，超越了任务特定专家模型和更大的通用模型。

## Abstract
ABACUS is a unified vision-language model that handles object counting, crowd counting, referring-expression counting, and count-faithful image generation without any benchmark-specific training required. Our model is built on existing 3B-parameter unified foundation model and is adapted for object localization tasks using three key innovations: density-aware adaptive zooming with objectness maps for spatial grounding; a boundary-aware count policy via GRPO to eliminate crop-boundary errors; and a cycle-consistent GRPO strategy where the understanding branch self-critiques generated outputs, closing the understanding-generation gap without any external annotations. ABACUS achieves state-of-the-art results across seven benchmarks, outperforming both task-specific specialists and larger generalist models.

## 1. 核心问题与研究动机

- **背景**：目标计数、人群计数、指代表达式计数（REC）以及计数条件图像生成是计算机视觉中的重要任务，但长期以来被独立研究，各自使用专门的架构、损失函数和训练流程，缺乏跨场景泛化能力。
- **核心挑战**：
  - 现有视觉语言模型（VLM）在稠密场景中计数精度低，缺乏空间定位能力。
  - 统一多模态模型（UMM）虽能同时处理理解与生成，但存在“协同差距”（synergy gap）：即使理解分支能正确计数，生成分支却无法产生准确数量的物体。
  - 没有外部标注或批评器来弥合此差距。
- **目标**：构建一个统一的模型，无需任何基准特定训练，即可同时处理目标计数、人群计数、指代计数和计数忠实图像生成，且优于任务专用专家和更大的通用模型。

## 2. 方法论

### 核心思想
基于现有3B参数统一基础模型（UniLIP-3B），通过三项创新适配目标定位与生成：

#### (1) 密度感知自适应缩放（Density-aware Adaptive Zooming）
- **问题**：MLLM在稀疏区域计数更可靠，稠密场景性能差。
- **方法**：使用冻结的GroundingDINO骨干网络 + 轻量MLP头作为密度指示器ϕ(·)，判断图像区域是否稠密（sd ≥ 0.5）。若为稠密，则递归分割为2×2子区域，直到达到最小分辨率γ。每个子区域由MLLM独立计数，局部计数聚合为全局计数。

#### (2) 注入目标性（Infusing Objectness in MLLM）
- **问题**：MLLM缺乏精确目标定位能力，直接输出坐标易产生幻觉。
- **方法**：从MLLM的多头自注意力（MHSA）层分解出每个视觉令牌的目标性分数ol,i(v)（通过ℓ2范数）。进一步推导空间注意力分布ql(v)，用高斯平滑的GT点标注通过Lobj损失监督，使模型内部化实例感知的空间结构。

#### (3) 边界感知计数策略（Boundary-aware Count Policy via GRPO）
- **问题**：自适应缩放导致物体位于裁剪边界，造成重复计数或遗漏。
- **方法**：对稠密图像划分2×2网格后，策略πθ输出结构化结果，将每个物体分类为内部（interior）、边缘（edge）或边界（boundary）。通过GRPO优化三个嵌套奖励：局部准确性（Δqr）、边界一致性（Δbr）、全局连贯性（Δgr）。奖励函数为soft惩罚：Δr = exp(-|pred - GT|/(GT + ε))。组相对优势计算后，策略通过最大化带KL惩罚的LGRPO目标更新。

#### (4) 循环一致GRPO（Cycle-consistent GRPO）
- **问题**：生成分支与理解分支之间存在协同差距，需要无外部批评器的自我改进。
- **方法**：给定计数条件提示t，生成分支采样K个候选图像x̂i。冻结的理解分支Mθ对每个候选图像计数ĉi，计算计数偏差奖励Rcnt = exp(-|ĉi - c*|/(c* + ε))。结合美学评分器Saes，得到复合奖励R = λc Rcnt + λa Saes。GRPO仅更新生成分支LoRA（πref固定），理解分支保持冻结。这样生成分支会逐渐被引导产生更忠实于目标计数的图像，且美学质量不下降。

### 训练策略（三阶段）
1. **阶段1（理解分支训练）**：使用密度感知缩放图像微调MLLM，损失Lund = LSFT + Lobj。后训练使用边界感知GRPO（2K步）。连接器和生成分支冻结。
2. **阶段2（连接器训练）**：冻结理解分支和DiT，训练连接器对齐MLLM特征与DiT条件空间。
3. **阶段3（生成分支训练）**：冻结MLLM，训练连接器和DiT。SFT后使用循环一致GRPO（5K步）改进计数准确性，仅更新生成分支LoRA。
