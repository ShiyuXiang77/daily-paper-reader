## 摘要
大多数同时支持视觉理解和图像生成的统一大型多模态模型（LMM）仍然依赖于精心策划的后训练监督，例如人工注释、偏好标签或外部奖励模型。我们探究一个统一的LMM是否能够仅使用未标注的图像自主提升这两种能力。我们提出了一种自进化的训练框架，包含三个内部角色：一个生成视觉问题的提议者，一个回答问题并评估它们的求解者，以及一个合成图像的生成者。训练仅使用自导出的一致性信号，无需人工注释、偏好标签或经过任务训练的外部奖励/判断模型。为了稳定学习，我们引入了求解者令牌熵（STE），这是一种基于令牌级预测不确定性的连续难度信号，即使在样本级一致性变得不可靠时仍然有用。对于图像生成，我们设计了一种多尺度内部评估方案，将问答保真度评分与循环一致性描述相结合。这创建了一种由求解者介导的耦合，其中更好的视觉理解使得更可靠的生成评估和更强的内部训练信号成为可能。该框架在基于扩散的BLIP3o、修正流BAGEL和自回归VARGPT-v1.1架构中保持相同的角色分解、奖励逻辑和训练计划，仅需要每个骨干网络的原生提示和生成接口。在八项理解指标上，我们的方法持续优于相应的基础模型。在BAGEL上，它在MMMU上实现了+3.5%的绝对增益，并将GenEval图像生成性能从82%提升至85%。代码和模型已公开发布。

## Abstract
Most unified large multimodal models (LMMs) that support both visual understanding and image generation still rely on curated post-training supervision, such as human annotations, preference labels, or external reward models. We ask whether a unified LMM can improve both abilities autonomously using only unlabeled images. We propose a self-evolving training framework with three internal roles: a Proposer that generates visual questions, a Solver that answers and evaluates them, and a Generator that synthesizes images. Training uses only self-derived consistency signals, without human annotations, preference labels, or task-trained external reward/judge models. To stabilize learning, we introduce Solver Token Entropy (STE), a continuous difficulty signal based on token-level prediction uncertainty that remains useful even when sample-level consistency becomes unreliable. For image generation, we design a multi-scale internal evaluation scheme that combines question-answer fidelity scoring with cycle-consistent captioning. This creates a solver-mediated coupling, where better visual understanding enables more reliable generation assessment and stronger internal training signals. The framework preserves the same role decomposition, reward logic, and training schedule across diffusion-based BLIP3o, rectified-flow BAGEL, and autoregressive VARGPT-v1.1 architectures, requiring only each backbone's native prompting and generation interface. Across eight understanding metrics, our method consistently improves over the corresponding base models. On BAGEL, it achieves a $+3.5\%$ absolute gain on MMMU and improves GenEval image generation performance from $82\%$ to $85\%$. Code and models are publicly released.

## 2. 论文提出的方法论

### 核心思想
- 将冻结的大型多模态骨干模型（backbone）分解为三个轻量级 **LoRA 适配器**（角色）：
  - **Proposer（提议器）**：针对图像生成视觉问题。
  - **Solver（求解器）**：回答问题并担任评估者。
  - **Generator（生成器）**：根据文本提示生成图像。
- 训练信号完全来自模型自身的**自一致性**和**令牌级不确定性**，形成两个交替循环：
  - **理解循环**：Proposer 提出挑战性问题 → Solver 在多种提示扰动下回答 → 通过自一致性熵和求解器令牌熵（STE）生成奖励，优化 Proposer 和 Solver。
  - **生成循环**：Generator 生成候选图像 → Solver 作为内部评估器，通过 QA 保真度评分和循环一致性描述来计算奖励，优化 Generator。
- 两个循环通过 Solver 相互耦合：更好的理解能力使 Solver 成为更可靠的生成评估者，从而反馈到生成环节。

### 关键技术细节
- **Solver Token Entropy (STE)**：计算求解器回答前 K 个令牌的最大 softmax 熵，经滚动窗口归一化为难度分数，弥补样本级自一致性熵可能坍缩的问题，提供连续的课程学习信号。
- **生成评估**：
  - **QA fidelity**：对生成的图像提出与真实图像相同的问题，比较答案一致性。
  - **Cycle-consistent captioning**：对生成图像进行描述，与原提示进行嵌入相似度计算，并添加图像-文本跨模态一致性。
  - 结合多样性奖励和矛盾惩罚。
- **优化方法**：
  - Proposer 使用 GRPO（组相对策略优化）；
  - Solver 使用 REINFORCE；
  - Generator 根据骨干类型使用 token-policy 更新（自回归）或奖励加权的去噪更新（扩散/流）。
  - 均加入 KL 正则化防止偏离冻结的参考模型。

### 算法流程（文字说明）
1. 从未标注图像池中采样图像。
2. **理解步骤**：Proposer 生成 K 个候选问题；Solver 在 N 种提示扰动下回答；计算自一致性熵和 STE；组合 Proposer 奖励和 Solver 奖励；更新 Proposer 和 Solver 适配器。
3. **生成步骤**：Solver 对真实图像进行描述作为生成提示；Proposer 生成诊断问题；Solver 在真实图像上获得参考答案；Generator 生成 L 个候选图像；计算 QA 保真度、循环一致性等奖励；更新 Generator 适配器。
4. 交替执行，默认比例为 3:2（理解:生成），约 10k 步。

## 7. 优点

- **完全无监督**：无需任何人工标注、偏好数据、外部奖励/判断模型，仅使用未标注图像。
- **跨架构通用**：一种固定配方（角色分解、奖励定义、调度）适用于扩散、修正流、自回归三种不同生成范式的骨干，仅需骨干的原生接口。
- **独特的 STE 设计**：从令牌级不确定性中提取连续难度信号，解决了样本级自一致性熵可能退化的难题，实现冷启动的课程学习。
- **不对称耦合设计**：理解提升促使求解器成为更优秀的内部评估器，从而驱动生成改善，而非对称梯度交换，更稳定且可解释。
- **轻量级**：仅训练少量 LoRA 适配器（约 0.8% 参数），冻结骨干，避免全微调过拟合或遗忘原能力。
