## 摘要
近期图像生成器在单图像生成和编辑中展现出惊人的逼真度和指令跟随能力。然而，受限于架构，它们无法实现交叠生成（文本-图像序列），而后者在视觉叙事、引导和具身操作中具有关键应用。即使是最新的开源统一多模态模型（UMMs）在这方面也表现有限。本文提出InterleaveThinker，这是首个旨在赋予任何现有图像生成器交叠生成能力的多智能体流水线。具体而言，我们使用规划智能体来组织图像-文本输入序列，指导图像生成器每个步骤所需执行的操作。随后，引入评判智能体评估生成器的输出，识别偏离规划指令的样本，并细化指令以重新生成。为实现该流水线，我们构建了Interleave-Planner-SFT-80k和Interleave-Critic-SFT-112k以进行格式冷启动。然后，我们开发了Interleave-Critic-RL-13k，利用GRPO在生成轨迹内强化逐步指令修正能力。由于单个交叠生成轨迹可能涉及超过25次生成器调用，优化整个轨迹在计算上不切实际。因此，我们提出准确率奖励和逐步奖励，使单步强化学习能有效引导整个生成轨迹。结果表明，InterleaveThinker提升了多种图像生成器的性能。在交叠生成基准上，其性能与Nano Banana和GPT-5相当。令人惊讶的是，它在基于推理的基准上也显著增强了基础模型；例如，在4步FLUX.2-klein上，我们在WISE和RISE上观察到显著提升。

## Abstract
Recent image generators have demonstrated impressive photorealism and instruction-following capabilities in single-image generation and editing. However, constrained by their architectures, they cannot achieve interleaved generation (text-image sequence), which has crucial applications in visual narratives, guidance, and embodied manipulation. Even the latest open-source Unified Multimodal Models (UMMs) exhibit limited performance in this regard. In this paper, we introduce InterleaveThinker, the first multi-agent pipeline designed to endow any existing image generator with interleaved generation capabilities. Specifically, we employ a planner agent to organize the image-text input sequence, instructing the image generator on the required execution at each step. Subsequently, we introduce a critic agent to evaluate the generator's outputs, identify samples that deviate from the planned instructions, and refine the instructions for regeneration. To implement this pipeline, we construct the Interleave-Planner-SFT-80k and Interleave-Critic-SFT-112k to perform a format cold-start. Then we develop Interleave-Critic-RL-13k to reinforce the step-wise instruction correction capability within a generation trajectory using GRPO. Since a single interleaved generation trajectory may involve over 25 generator calls, optimizing the entire trajectory is computationally impractical. Therefore, we propose accuracy reward and step-wise reward, allowing single-step RL to effectively guide the entire generation trajectory. The results show that InterleaveThinker improves performance across various image generators. On interleaved generation benchmarks, it achieves performance comparable to Nano Banana and GPT-5. Surprisingly, it also significantly enhances the base model on reasoning-based benchmarks; for example, on 4-step FLUX.2-klein, we observe substantial gains on WISE and RISE.

## 1. 核心问题与整体含义
- **研究动机**：现有图像生成器（如 FLUX、Stable Diffusion）虽然单图像生成与编辑能力强大，但受限于输出架构（仅输出图像），无法直接生成文本与图像的交替序列（即**交叠生成**）。统一多模态模型（UMM，如 Emu3.5、Janus-Pro）虽原生支持交叠生成，但在长序列任务中存在两个致命缺陷：**视觉过度依赖**（模型被中间视觉状态误导而过早终止）和**逐步骤误差累积**（每一步的质量下降逐步放大，最终破坏整体输出）。
- **整体含义**：本文旨在通过多智能体解耦的方式，让任意固定的图像生成器“外包”规划与纠错能力，从而获得稳健的、长程的交叠生成能力，并首次将强化学习应用于此类多智能体长轨迹优化。

## 2. 方法论：核心思想与关键技术
- **核心思想**：将交叠生成分解为三个模块——规划器（Planner）、生成器（Generator，冻结）、评判器（Critic），形成闭环流水线。规划器预先输出全局指令序列，避免中间视觉反馈干扰；评判器在每步执行后评估生成图像与指令的偏差，必要时生成细化提示词重新生成。
- **关键技术细节**：
  - 规划器接收交叠输入序列 \(S\)，输出 \(N\) 步的计划：每一步包含步骤指令 \(u_i\)、模型友好提示词 \(p_i\)、辅助文本 \(a_i\)。
  - 生成器根据当前提示词和前一步图像生成新图像 \(I_i^t = \text{Generator}(r_i^t, I_{i-1})\)。
  - 评判器评估前后图像与原始指令，输出二元判断 \(j_i^t\)、细化提示词 \(r_i^{t+1}\) 和推理过程。循环直到判断为成功或达到最大迭代 \(T_{\text{max}}\)。
  - **训练数据构建**：通过分层分类法（8主类→75子类→30+领域词库→100+模板）生成约4万文本提示词；使用 Gemini 2.5 Pro 和 Nano Banana Pro 生成多智能体轨迹；经严格过滤后得到三个数据集：
    - `Interleave-Planner-SFT-80k`：用于规划器监督微调。
    - `Interleave-Critic-SFT-112k`：用于评判器冷启动。
    - `Interleave-Critic-RL-13k`：用于评判器的 GRPO 强化学习。
  - **双奖励策略**：由于单轨迹可能超过25步生成调用，直接端到端RL不现实。作者提出单步RL：
    - **准确率奖励** \(R_{\text{acc}} = -|\hat{J} - J|\)，衡量评判器的二分类正确性。
    - **逐步奖励** \(R_{\text{step}} = \text{score}(I^{t+1}) - \text{score}(I^t)\)，衡量细化提示词是否带来改善（使用专家模型 Gemini 2.5 Pro 评分）。
    - 最终奖励 \(R = 0.5R_{\text{format}} + 0.5(\alpha R_{\text{acc}} + (1-\alpha)R_{\text{step}})\)，\(\alpha=0.2\)。
  - 采用 GRPO 算法，在组内归一化优势后更新评判器策略。

## 6. 主要结论与发现
- InterleaveThinker 成功让**任意冻结的图像生成器**获得交叠生成能力，在 UEval 上达到与 Nano Banana Pro（76.1）接近的分数（66.3/67.2），大幅超越所有开源 UMM（最高 49.1）。

## 7. 优点
- **架构创新**：首个将多智能体流程（规划→生成→评判）应用于图像交叠生成的工作，有效解决了 UMM 的视觉过度依赖和误差累积问题。
- **训练高效**：提出双奖励实现单步 RL 等效于全轨迹优化，大幅降低了计算开销，使得长序列 RL 可行。
- **数据系统**：设计了自顶向下的提示词生成和轨迹构造流水线，并引入基于评分方差的 SFT/RL 数据拆分策略，确保数据质量。
- **模型无关性**：框架可与任何图像生成/编辑模型搭配，在 FLUX.2 和 Qwen-Image-Edit 上均验证有效。
- **实验全面**：在多个基准上对比了广泛的方法，消融实验覆盖关键组件，结论可靠。

## 8. 不足与局限
- **受限于基础生成器**：框架无法生成基础模型训练集外的新概念（例如图8中FLUX.2-klein不认识的“Songling Aloha”风格导致颜色偏移）。
- **计算开销仍较大**：虽然单步RL代价低，但推理时每步可能多次调用生成器（最多5次），总延迟较高。论文未提供与UMM的直接推理时间对比。
- **泛化性验证不足**：仅测试了两种基础生成器（FLUX.2-klein 和 Qwen-Image-Edit），未在更多流行模型（如 SD3.5、MJD）上实验。
- **规划器未用RL**：作者提到因轨迹太长、奖励稀疏，未对规划器使用RL，可能限制了规划质量的进一步提升。
- **仅针对图像生成**：方法虽通用，但论文只涉及静态图像的交叠生成，未扩展到视频、3D或交互式场景。
