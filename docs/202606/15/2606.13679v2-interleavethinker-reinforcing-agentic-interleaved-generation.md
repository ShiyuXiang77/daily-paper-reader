## 摘要
最近的图像生成器在单图像生成和编辑中展现了令人印象深刻的逼真度和指令遵循能力。然而，受其架构限制，它们无法实现交错生成（文本-图像序列），而这一能力在视觉叙事、指导和具身操作中具有关键应用。即使是最新的开源统一多模态模型（UMMs）在这方面也表现有限。在本文中，我们引入了InterleaveThinker，这是首个旨在赋予任何现有图像生成器交错生成能力的多智能体管道。具体来说，我们采用一个规划器智能体来组织图像-文本输入序列，指导图像生成器在每一步所需的执行内容。随后，我们引入一个评论家智能体来评估生成器的输出，识别偏离规划指令的样本，并细化指令以重新生成。为了实现该管道，我们构建了Interleave-Planner-SFT-80k和Interleave-Critic-SFT-112k来执行格式冷启动。然后我们开发了Interleave-Critic-RL-13k，使用GRPO在生成轨迹内强化逐步指令修正能力。由于单个交错生成轨迹可能涉及超过25次生成器调用，优化整个轨迹在计算上不切实际。因此，我们提出了准确率奖励和逐步奖励，使得单步RL能够有效引导整个生成轨迹。结果表明，InterleaveThinker在各种图像生成器上提升了性能。在交错生成基准上，它达到了与Nano Banana和GPT-5相当的性能。令人惊讶的是，它还在基于推理的基准上显著提升了基础模型；例如，在4步FLUX.2-klein上，我们在WISE和RISE上观察到了显著提升。

## Abstract
Recent image generators have demonstrated impressive photorealism and instruction-following capabilities in single-image generation and editing. However, constrained by their architectures, they cannot achieve interleaved generation (text-image sequence), which has crucial applications in visual narratives, guidance, and embodied manipulation. Even the latest open-source Unified Multimodal Models (UMMs) exhibit limited performance in this regard. In this paper, we introduce InterleaveThinker, the first multi-agent pipeline designed to endow any existing image generator with interleaved generation capabilities. Specifically, we employ a planner agent to organize the image-text input sequence, instructing the image generator on the required execution at each step. Subsequently, we introduce a critic agent to evaluate the generator's outputs, identify samples that deviate from the planned instructions, and refine the instructions for regeneration. To implement this pipeline, we construct the Interleave-Planner-SFT-80k and Interleave-Critic-SFT-112k to perform a format cold-start. Then we develop Interleave-Critic-RL-13k to reinforce the step-wise instruction correction capability within a generation trajectory using GRPO. Since a single interleaved generation trajectory may involve over 25 generator calls, optimizing the entire trajectory is computationally impractical. Therefore, we propose accuracy reward and step-wise reward, allowing single-step RL to effectively guide the entire generation trajectory. The results show that InterleaveThinker improves performance across various image generators. On interleaved generation benchmarks, it achieves performance comparable to Nano Banana and GPT-5. Surprisingly, it also significantly enhances the base model on reasoning-based benchmarks; for example, on 4-step FLUX.2-klein, we observe substantial gains on WISE and RISE.

## 1. 核心问题与整体含义（研究动机和背景）
现有图像生成器（如扩散模型）在单图生成上表现优异，但受限于架构，无法原生支持文本‑图像序列的**交错生成**（interleaved generation）。统一多模态模型（UMM）虽理论上支持，但在长序列任务中存在两大关键问题：  
- **视觉过依赖**：模型过于依赖前一步的视觉状态，容易提前终止在看似接近目标但实际未完成的中间状态。  
- **步骤级误差累积**：没有有效的自校正机制，早期微小错误会逐步放大，最终破坏输出质量。  

这些缺陷严重限制了交错生成在视觉叙事、操作指导、具身机器人等实际场景中的应用。本文提出首个多智能体框架**InterleaveThinker**，通过解耦规划与评价，为任意冻结的图像生成器赋予强大的交错生成能力。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
将交错生成分解为三个独立模块：**规划器（Planner）**、**生成器（Generator）**、**评论家（Critic）**。规划器预先产生全部步骤的指令，避免中间视觉反馈干扰；生成器按指令生成/编辑图像；评论家评估每步输出，若不合格则细化提示词，驱动重生成。

### 关键技术细节
- **Planner**：基于 Qwen3‑VL‑8B，通过 SFT 学习将用户输入转化为 N 步执行计划，每步包含 `(instruction, prompt, auxiliary_text)`。纯文本提示规划使用特定系统提示，多模态输入（已有图像序列）使用另一套系统提示。
- **Critic**：同样初始化自 Qwen3‑VL‑8B，通过 SFT 学习评价格式，输出 `<think>... </think><answer>{ "previous_step_success": boolean, "refine_prompt": "..."}</answer>`。
- **Generator**：可选用任意现成图像生成/编辑模型，如 FLUX.2‑klein‑9B 或 Qwen‑Image‑Edit。
- **训练数据构造**：  
  1. 用 8 大类（～75 子类）模板 + 30+ 领域词库生成约 40k 文本提示。  
  2. 调用 Gemini 2.5 Pro 和 Nano Banana Pro 生成完整多智能体轨迹，经严格过滤（步骤内评分变化、迭代判断平衡）得到三个数据集：  
     - `Interleave-Planner-SFT-80k`（规划器 SFT）  
     - `Interleave-Critic-SFT-112k`（评论家 SFT）  
     - `Interleave-Critic-RL-13k`（评论家 RL 强化）  
- **强化学习（RL）**：使用 GRPO 算法，但为克服全轨迹 RL 计算不可行，提出了**双重奖励**策略：  
  - **准确率奖励 (R_acc)**：鼓励 Critic 正确预测二值判断。  
  - **步骤奖励 (R_step)**：基于新生成图像与原图像的评分差异，奖励有效的提示细化。  
  最终奖励组合：`R = 0.5 * R_format + 0.5 * (α R_acc + (1‑α) R_step)`，α=0.2。通过单步 RL 实现轨迹级对齐。

### 公式/算法流程（文字说明）
1. Planner 对输入 S 输出 N 步计划 `{(u_i, p_i, a_i)}`。
2. 对每一步 i：  
   - 迭代 t=0..T_max，Generator 根据当前提示 r^t_i 和上一步图像 I_{i-1} 生成候选 I^t_i。  
   - Critic 评估 I_{i-1} → I^t_i 的变化，输出判断 j^t_i、新提示 r^{t+1}_i 及推理过程。  
   - 若 j = True 或达到 T_max，则接受图像，输出 `(I_i, a_i)`，进入下一步。

## 3. 实验设计

## 6. 主要结论与发现
1. **InterleaveThinker 可赋予任意冻结图像生成器强大的交错生成能力**，在 UEval 和 CoMM 上达到甚至超过专有模型水平。
