## 摘要
多模态情感分析（MSA）从语言、声学与视觉信号推断人类情感。近期方法日益通过生成式读取（generative readout）适配大型多模态模型（LMM）：提示模型以文本字符串形式输出情感分数。尽管方便，但这将连续回归与离散自回归解码绑定，带来了未量化的成本。我们重新审视该读取机制，并基于原生全模态LLM（Qwen2.5-Omni-7B）的Thinker模块提出一种判别性公式。我们不对文本进行解码，而是通过轻量级回归头，在单次前向传播中将最后一个非填充token的最终层隐状态映射为连续分数。使用4比特量化与低秩适应（QLoRA），整个7B管线（包括视频和音频处理）可在单个消费级GPU（RTX 5090, 32 GB）上训练，峰值内存10–21 GB，可训练参数占比1.14%。通过固定主干网络、数据和LoRA配置的受控比较，我们分离出读取方式的影响。在CMU-MOSI和CMU-MOSEI上，我们的判别式读取在没有任务特定特征工程的情况下达到了最先进的准确率（MOSI: MAE 0.551, Corr 0.888; MOSEI: MAE 0.506, Corr 0.790），并展现出强大的多随机种子稳定性。相比之下，生成式读取——即使经过等效的监督训练——平均绝对误差增加一倍以上，产生无法解析或超出范围的输出（零样本下2.8%），且延迟更高。模态消融实验揭示了CMU-MOSI上的文本主导模式。我们的发现表明，LLM的读取方式与其训练方式同样重要，证明判别式读取为连续MSA提供了一种更准确、高效且可靠的替代方案。

## Abstract
Multimodal sentiment analysis (MSA) infers human affect from language, acoustic, and visual signals. Recent methods increasingly adapt large multimodal models (LMMs) via generative readout: prompting the model to emit a sentiment score as a text string. While convenient, this ties continuous regression to discrete autoregressive decoding, incurring unmeasured costs. We revisit this readout mechanism and propose a discriminative formulation built on the Thinker module of a native omni-modal LLM (Qwen2.5-Omni-7B). Instead of text decoding, we map the final-layer hidden state of the last non-padding token to a continuous score via a lightweight regression head in a single forward pass. Using 4-bit quantization and low-rank adaptation (QLoRA), the entire 7B pipeline -- including video and audio processing -- trains on a single consumer GPU (RTX 5090, 32 GB) with 10-21 GB peak memory and 1.14% trainable parameters. Through a controlled comparison fixing the backbone, data, and LoRA configuration, we isolate the impact of the readout. On CMU-MOSI and CMU-MOSEI, our discriminative readout reaches state-of-the-art accuracy without task-specific feature engineering (MOSI: MAE 0.551, Corr 0.888; MOSEI: MAE 0.506, Corr 0.790) and exhibits strong multi-seed stability. In contrast, the generative readout -- even after equivalent supervised training -- more than doubles the mean absolute error, yields unparsable or out-of-range outputs (2.8% zero-shot), and suffers from higher latency. Modality ablations reveal a text-dominant regime on CMU-MOSI. Our findings indicate that how an LMM is read out is as consequential as how it is trained, demonstrating that a discriminative readout offers a more accurate, efficient, and reliable alternative for continuous MSA.

- **背景**：多模态情感分析（MSA）旨在从语言、声学、视觉信号中推断连续情感分数（如 \([-3, +3]\)）。近期主流方法基于大型语言模型（LLM）或大型多模态模型（LMM），通常采用 **生成式读取（generative readout）**：提示模型输出情感分数文本字符串（例如 "1.5"），再解析为数值。
- **核心问题**：生成式读取存在根本性不匹配——将连续的回归目标强制通过离散的自回归解码通道来输出，这会导致精度受限、延迟增加，且可能出现不可解析或超范围的输出。然而，这种成本在文献中几乎未被量化，生成式读取几乎成为默认选择。
- **研究意义**：本文首次将读取机制（readout）视为LLM-based MSA中的一阶设计选择，并通过严格受控实验证明：**对于连续MSA，读取方式的选择与训练方式同等重要**；判别式读取（discriminative readout）在精度、效率和可靠性上均优于生成式读取。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：摒弃生成式解码头，直接从LLM内部隐藏状态中“读出”连续情感分数。具体而言：
    1. 采用 **Qwen2.5-Omni-7B** 的 Thinker 模块作为共享编码器，它原生处理文本、音频、视频，输出统一隐状态序列 \(H = [h_1, ..., h_T]\)。
    2. 取最后一个非填充（non-padding）token 的最后一层隐状态 \(z = h_\ell\)。
    3. 通过一个轻量级 MLP 回归头 \(g_\phi\) 直接预测标量 \(\hat{y}\)：  
       \[
       \hat{y} = g_\phi(z) = W_2\,\text{Dropout}(\text{ReLU}(\text{LN}(W_1 z + b_1))) + b_2
       \]
    4. 训练目标为归一化后的 MAE（L1 损失），并对标签进行标准化（减去训练集均值，除以标准差）以防止预测崩溃。
    5. 推理时只需一次前向传播，无需自回归解码，输出始终为有界实数。

- **关键技术细节**：
    - **参数高效微调**：使用 **QLoRA**（4-bit NF4 量化 + LoRA），LoRA 秩 \(r=32\)，应用于注意力与前馈投影（q, k, v, o, gate, up, down）。可训练参数约 103M（仅占 1.14%），冻结主干。
    - **高效多模态输入**：对视频片段均匀采样固定帧数（最多16帧），限制每帧像素预算；音频使用 DeepFilterNet 降噪；解码采用工作进程+超时保护，避免损坏片段挂起训练。
    - **受控对比设置**：为公平比较两种读取方式，固定同一骨干、同一数据、相同 LoRA 配置，仅改变读取方式（判别头 vs. 生成解码）。生成式读取训练时使用语言建模损失（对目标数字字符串进行交叉熵），推理时贪心解码并解析；记录不可解析率和超出范围率。

### 6. 论文的主要结论与发现

1. **读取方式至关重要**：在完全相同的骨干网络、数据和适应预算下，判别式读取的 MAE 比生成式读取低一半以上（0.667 vs. 1.443），相关系数高出近一倍（0.824 vs. 0.491），且训练生成式读取并不能缩小差距（反而使相关系数下降至 0.197）。
2. **判别式读取更可靠**：输出始终为有界实数，不可解析率和超范围率严格为零；而生成式读取零样本下有 2.8% 的不可解析输出和 0.05% 的超范围输出。
3. **判别式读取更高效**：单次前向传播即可得到结果，推理时间低于生成式读取（1.04s vs. 1.47s）。
4. **达到 SOTA 且无任务特定特征工程**：在 MOSI 和 MOSEI 上，判别式读取的 MAE 和 Corr 与最强近期方法持平或略优，且无需外部特征提取器和手工融合模块。
5. **隐状态已经按情感强度组织**：kNN 标签平滑度统计量显示，相邻隐状态的标签差异远低于随机基线（降低 49.9%），解释了为何轻量头即可有效读取。
6. **模态贡献因数据集而异**：在 CMU-MOSI 上文本主导，添加视觉/音频反而不利于回归精度；这被诚实讨论为数据集特性，而非方法缺陷。
