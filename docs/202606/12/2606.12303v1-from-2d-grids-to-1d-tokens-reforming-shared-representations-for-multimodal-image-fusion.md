## 摘要
多模态图像融合旨在将来自不同模态的互补信息整合到一幅融合图像中，该图像在保持全局一致外观的同时保留丰富的局部细节。现有方法在二维特征网格上构建共享表示，这擅长建模局部结构，但对图像级全局外观因素的利用有限。为了平衡这些目标，我们引入了一种基于冻结预训练图像令牌化器的紧凑一维令牌接口，用于建模非局部外观/基础因素。我们的设计不是将令牌化器用作重建主干，而是将一维令牌空间作为全局载体，同时保留用于局部结构恢复的二维空间路径。具体来说，我们引入了选择性令牌编辑（STE），它稀疏地更新/替换少量关键令牌，提供了一种轻量级机制来引导全局外观一致性，同时保持融合主干不变并避免额外损失。在四个常用基准上的实验表明，我们的方法实现了最佳整体性能，在全局一致性和局部保真度方面均具有一致的多指标改进。项目页面：https://zju-xyc.github.io/1D-Fusion-Project-Page/

## Abstract
Multimodal image fusion aims to integrate complementary information from different modalities into a fused image that preserves rich local details while maintaining globally consistent appearance. Existing approaches build shared representations on 2D feature grids, which excel at modeling local structures but offer limited leverage over image-level global appearance factors. To balance these objectives, we introduce a compact 1D token interface based on a frozen pretrained image tokenizer for modeling non-local appearance/base factors. Rather than using the tokenizer as a reconstruction backbone, our design uses the 1D token space as a global carrier while retaining the 2D spatial pathway for local structure restoration. Specifically, we introduce Selective Token Editing (STE), which sparsely updates/replaces a small set of critical tokens, providing a lightweight mechanism to steer global appearance coherence while keeping the fusion backbone unchanged and avoiding extra losses. Experiments on four commonly used benchmarks show that our method achieves the best overall performance, with consistent, multi-metric improvements in both global coherence and local fidelity. ## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：多模态图像融合（如红外-可见光融合、医学图像融合）需要在融合图像中同时保留丰富的局部细节和全局一致的外观（如亮度、对比度、色调）。然而，现有方法普遍在**二维（2D）密集特征网格**上构建共享表示，这种表示结构天然擅长建模局部结构，但**难以有效控制和协调图像级的全局外观因素**。
- **根本原因**：全局外观因素（如光照、对比度）是非空间索引的低维变量，而2D网格是空间索引的高维表示。在2D网格中，全局信息只能通过空间广播隐式分布于所有位置，导致与局部细节、模态特定噪声**纠缠**，使得全局外观调控在统计和优化上不稳定。
- **研究意义**：本文提出用**紧凑的一维（1D）令牌序列**替代2D网格作为共享表示载体，为全局外观提供独立、可控的接口，同时保留2D空间通路处理局部细节，从而平衡全局连贯性与局部保真度。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将融合中的**基础（base，全局外观）** 与**细节（detail，局部结构）** 显式解耦。基础因子由1D令牌空间承载，细节因子继续由2D空间通路处理。
- **关键技术细节**：
  - **1D令牌化器**：使用冻结的预训练**TiTok**（An Image is Worth 32 Tokens）作为紧凑的1D令牌生成器，将输入图像映射为32个12维的令牌序列。
  - **选择式令牌编辑（Selective Token Editing, STE）**：在1D令牌空间中识别少量对全局外观敏感的令牌位置（通过Gumbel-Softmax探针离线确定，最终固定为位置12和18的通道6-8），仅在这些位置添加可学习的偏移量（Δ），其余位置不变。这种方式轻量、稀疏，不改变融合主干，无需额外损失函数。
  - **令牌到图接口（Token-to-Map Mapping, π(·)）**：将1D令牌序列通过线性映射、残差聚合和三级上采样转换为与2D通路兼容的特征图（先得到32×32粗图，再逐级上采样至256×256，并在每一级注入来自原始图像的尺度对齐细节）。
  - **因式分解融合（Factorized Fusion）**：使用私有编码器将令牌诱导的特征图分解为基础图B和细节图D，然后分别在基础空间和细节空间进行独立融合（Bf = F_base(BV, BI), Df = F_detail(DV, DI)），并采用**残差解码**（预测残差ΔI，最终图像If = Iref + ΔI，其中Iref=IV+II）。
- **训练策略**：两阶段训练，令牌化器全程冻结。
  - **阶段一（重建热身）**：仅进行单模态重建，学习稳定分解（损失含SSIM、L2及分解正则项L_decomp，鼓励细节互补、基础一致）。
  - **阶段二（融合训练）**：激活融合模块，使用融合损失（L_fusion，包括强度最大化和梯度最大化L1损失）与分解正则项联合优化。

## 6. 主要结论与发现

- **主要结论**：将共享表示从2D网格改革为1D令牌空间，结合选择式令牌编辑，能够有效提升多模态图像融合的全局外观连贯性和局部保真度，在四个基准上取得最优或次优的整体性能，下游检测/分割任务也表现最佳。
- **关键发现**：
  - 1D令牌空间更适合承载全局外观因子，而2D通路更利于局部结构恢复。
  - 稀疏编辑少数令牌（如位置12和18）即可显著改善融合质量，说明全局因子在高度压缩的1D表示中高度集中。
  - TiTok作为1D接口优于识别导向编码器（DINOv3、CLIP），因为融合需要外观敏感性而非语义不变性。

## 7. 优点

- **创新性**：首次系统分析2D网格在全局外观调控上的结构性缺陷，并提出1D令牌接口作为可控全局载体的新视角。
- **轻量高效**：冻结令牌化器，仅1.3M可训练参数，STE仅编辑2个位置的3个通道，计算开销可控。
- **兼容性**：令牌-图接口设计使得该方法可直接嵌入现有2D融合骨干，无需大幅修改。
- **实验充分**：涵盖多个数据集、多种模态（红外-可见光、医学）、下游任务及充分消融，验证全面。

## 8. 不足与局限

- **实验覆盖**：未报告GPU型号和训练时间等算力细节，影响可复现性评估。医学融合实验仅在一个数据集（Harvard）上验证，且指标仅4个，泛化性待加强。
