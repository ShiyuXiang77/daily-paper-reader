## 摘要
我们提出Edit3DGS，一个用于动态3D头部编辑的统一框架，该框架将2D指令引导扩散与3D高斯溅射相结合。与分别处理基于帧的编辑或静态3D重建的现有方法不同，我们的方法将图像域中的语义可控性与逼真、时间一致的3D表示相耦合。给定输入视频，使用文本条件扩散模型对可编辑面部区域进行掩膜和修改，以支持精细操作，如表情变换、属性修改和外观细化。然后通过3D高斯溅射聚合编辑后的帧，生成连贯、高保真的虚拟形象，同时保留身份和运动动态。为增强一致性，Edit3DGS结合了多视图批量编辑和轻量级修复策略，以恢复跨时间步丢失的表情。实验结果表明，我们的框架实现了可控、无伪影的头部编辑，具有平滑的时间过渡，在虚拟化身、沉浸式通信、电影制作和交互媒体中具有实际应用价值。

## Abstract
We present Edit3DGS, a unified framework for dynamic 3D head editing that integrates 2D instruction-guided diffusion with 3D Gaussian splatting. Unlike prior approaches that separately address frame-based edits or static 3D reconstruction, our method couples semantic controllability in the image domain with photorealistic, temporally consistent 3D representations. Given an input video, editable facial regions are masked and modified using a text-conditioned diffusion model to support fine-grained operations such as expression transformation, attribute modification, and appearance refinement. The edited frames are then aggregated through 3D Gaussian splatting to produce a coherent, high-fidelity avatar that preserves both identity and motion dynamics. To enforce consistency, Edit3DGS incorporates multi-view batch editing and lightweight inpainting strategies that recover lost expressions across timesteps. Experimental results demonstrate that our framework enables controllable, artifact-free head editing with smooth temporal transitions, offering practical applications in virtual avatars, immersive communication, film production, and interactive media.

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 利用 2D 指令引导扩散模型（Instruct-Pix2Pix）对从 3DGS 头像渲染出的多视图图像进行语义编辑，再通过**高斯拟合（Gaussian fitting）** 将编辑后的图像聚合更新 3D 高斯模型，迭代优化实现整体编辑。关键挑战在于如何生成跨视图一致且随时间平滑的编辑图像。

### 关键技术细节
1. **基础模型**：采用 GaussianAvatars 作为基座模型，该模型将 3D 高斯基元绑定到 FLAME 网格三角形上，实现基于参数的表达式和姿态控制。
2. **编辑流程（render-edit-aggregate）**：
   - 从选定的摄像头和时间步渲染一组图像。
   - 使用指令引导的 2D 扩散模型（Instruct-Pix2Pix）对图像进行编辑。
   - 将编辑后的数据集作为输入，通过高斯拟合更新 3D 高斯模型。经过多次迭代，最终得到编辑后的 3D 头像。
3. **空间一致性编辑（Multi-view Batch Editing）**：
   - 参考 DGE 方法，将一组视图作为批量输入到 2D 扩散模型中。
   - **关键视图编辑**：在每个去噪步骤中，随机选取子集作为关键视图，利用空间-时间注意力模块同时编辑，使各视图融入上下文信息。
   - **特征注入**：基于极线约束确定关键视图与非关键视图之间的对应关系，将关键视图的特征传播到所有视图，确保批量内一致性。
4. **时间一致性编辑（Inpainting 策略）**：
   - 观察到 Instruct-Pix2Pix 在处理不同时间步的渲染图时，往往丢失复杂表情（尤其眼睛和嘴巴周围细节）。
   - **解决方案**：在编辑过程中，利用 SAM 2 + Grounding DINO 自动生成眼睛和嘴巴区域的掩膜，将这些区域的潜在向量在去噪后期重新注入，以恢复并保留原始表情。同时支持局部编辑。
5. **训练与推理**：
   - 训练：Adam 优化器，学习率 1e-2，每编辑运行 `10 × n` 次迭代（n 为时间步/批次数量）。
   - 推理：可直接通过 FLAME 参数驱动编辑后的头像生成新姿态、表情和视角的渲染。
