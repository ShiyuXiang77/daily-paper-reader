## 摘要
图像内机器翻译（IIMT）旨在翻译图像中的场景文本，并将翻译后的文本渲染回原始区域，同时保持整体视觉外观。最近统一的多模态模型通过将视觉-文本理解和图像生成结合在单一框架内，提供了一种有前景的解决方案。然而，直接将这些模型应用于IIMT仍然具有挑战性。具体而言，它们常常面临理解-生成冲突（理解阶段推断的翻译与生成阶段使用的文本监督不一致）以及空间位置错位（渲染文本无法精确匹配目标文本区域）的问题。为解决这些问题，我们提出了UniTranslator，一个将翻译理解和文本编辑紧密耦合的统一多模态IIMT框架。具体来说，我们引入了理解-生成对齐模块（UGAM），以弥合理解和生成之间的表示差距，促进翻译内容预测与文本渲染之间的语义一致性。我们进一步提出了空间掩码解码器（SMD），通过对文本区域进行像素级监督，改善生成过程中的空间定位、几何对齐和布局可控性。在多个基准上的大量实验表明，UniTranslator在多种语言方向和复杂真实布局上均取得了最先进的性能。此外，我们的结果揭示了翻译理解与图像生成之间存在强烈的相互增强效应，突显了统一翻译多模态学习的优势。代码见 https://github.com/SeerRay-Lab/Unitranslator。

## Abstract
In-Image Machine Translation (IIMT) aims to translate scene text in an image and render the translated text back into the original regions while preserving the overall visual appearance. Recent unified multimodal models provide a promising solution by combining visual-text understanding and image generation within a single framework. However, directly adapting such models to IIMT remains challenging. In particular, they often suffer from understanding-generation conflicts, where the translation inferred during understanding is inconsistent with the text supervision used in generation, and spatial position misalignment, where the rendered text does not accurately match the target text regions. To address these issues, we present UniTranslator, a unified multimodal framework for IIMT that tightly couples translation understanding and text editing. Specifically, we introduce an Understand-Generation Alignment Module (UGAM) to bridge the representation gap between understanding and generation, encouraging semantic consistency between translated content prediction and text rendering. We further propose a Spatial Mask Decoder (SMD) with pixel-level supervision over text regions to improve spatial grounding, geometric alignment, and layout controllability during generation. Extensive experiments on multiple benchmarks demonstrate that UniTranslator achieves state-of-the-art performance across diverse language directions and complex real-world layouts. Moreover, our results reveal a strong mutual reinforcement effect between translation understanding and image generation, highlighting the advantage of unified translation multimodal learning. Code is available at https://github.com/SeerRay-Lab/Unitranslator.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **任务定义**：图像内机器翻译（In-Image Machine Translation, IIMT）要求将图像中的场景文本翻译成目标语言，并维持原图的视觉外观（背景、布局、字体风格等）。该任务在旅行辅助、增强现实、跨境电商等场景具有重要应用价值。
- **现有挑战**：
  - **理解-生成冲突（Understanding-Generation Conflicts）**：统一多模态模型（UMMs）中的理解分支（翻译预测）与生成分支（图像渲染）接受不一致的监督信号：理解分支学习语义正确的翻译，但生成分支必须再现具体的视觉文本实例，导致语义歧义与像素级渲染目标互相矛盾。
  - **空间位置错位（Spatial Position Misalignment）**：通用图像编辑模型缺乏对文本区域的精细空间控制，常将翻译文本放置在错误区域或布局错位，破坏了图像的结构完整性。
- **研究动机**：现有的级联管道（OCR+MT+编辑）存在误差累积；专有UMMs如GPT-Image-1性能较好但不可开源；开源UMMs（如Nano-Banana、Qwen-Image）直接用于IIMT效果差。因此，需要针对UMMs有效解锁其IIMT能力，同时解决上述两个关键瓶颈。

## 2. 论文提出的方法论

- **核心思想**：将翻译理解与视觉文本编辑在单一统一框架内联合建模，通过显式模块弥合理解与生成的表示鸿沟，并引入像素级空间监督提升位置精度。
- **总体架构**（图2）：基于Qwen2.5-VL + MMDiT（扩散Transformer）。输入源图像和指令，多模态骨干提取共享表示，然后分三支：理解分支（自回归解码预测翻译文本）、生成分支（扩散生成目标图像）、掩码分支（预测文本区域掩码）。
- **关键技术细节**：
  1. **理解-生成对齐模块（UGAM）**：  
     将理解表示 \(E_{und}\) 作为Query，多模态表示 \(E_{VL}\) 作为Key/Value，通过交叉注意力+MLP生成对齐条件嵌入 \(E_c\)。该嵌入作为生成分支的软条件，而非硬性文本约束，缓解语义到像素的冲突。
  2. **空间掩码解码器（SMD）**：  
     以视觉表示 \(E_V\) 为Key/Value，理解表示 \(E_{und}\) 为Query，经交叉注意力融合后，通过多层反卷积预测文本区域掩码 \(\hat{M}_{pred}\)。训练使用二元交叉熵+Dice损失组合（\(\lambda_{bce}=1,\lambda_{dice}=1\)）。
  3. **两阶段训练策略**：
     - 阶段一（模块预热）：冻结预训练骨干和扩散生成器，仅优化UGAM和SMD，训练70个epoch，学习率1e-4，batch size 32。
     - 阶段二（联合微调）：开启LoRA（rank=64）微调Qwen2.5-VL，同时更新UGAM、SMD和MMDiT注意力投影层。训练30个epoch，余弦学习率衰减（峰值1e-5，warm-up 100步），batch size 8。
  4. **损失函数**：  
     \( \mathcal{L}_{total} = 0.1 \mathcal{L}_{und} + 1 \cdot \mathcal{L}_{gen} + 0.1 \cdot \mathcal{L}_{mask} \)，其中 \(\mathcal{L}_{und}\) 为交叉熵文本预测损失，\(\mathcal{L}_{gen}\) 为扩散velocity预测损失，\(\mathcal{L}_{mask}\) 为掩码BCE+Dice损失。
  5. **推理流程**：理解分支自回归生成预测文本 \(T_{pred}\)，代替阶段一中的教师强制（ground-truth \(T_{gt}\)），UGAM基于预测文本生成条件嵌入，再由生成分支和SMD协同输出最终图像。
