## 摘要
多模态大语言模型（MLLMs）通常采用基于重采样的投影器，将稠密视觉特征转换为紧凑的token序列以进行语言建模。现有的大多数重采样器通过全局交叉注意力采用单一的固定聚合范围，这可能模糊细粒度的局部证据，并限制在固定token预算内捕获局部细节和全局上下文的能力。在这项工作中，我们提出了MS-Resampler，一种用于MLLMs的多范围视觉重采样框架。MS-Resampler通过将显式的空间范围先验注入重采样注意力中，实例化了多个特定范围的重采样器，使每个分支能够以从局部到全局的特定粒度聚合视觉信息。然后，这些特定范围的重采样器的输出被自适应地融合，以生成用于语言建模的最终视觉表示。在十个公共多模态基准上的大量实验表明，与传统的单范围重采样器相比，MS-Resampler在视觉理解和多模态推理方面持续改进，同时仅引入极小的计算开销。

## Abstract
Multimodal large language models (MLLMs) typically employ resampling-based projectors to transform dense visual features into a compact token sequence for language modeling. Most existing resamplers adopt a single, fixed aggregation scope via global cross-attention, which can blur fine-grained local evidence and limit the ability to capture both local details and global context within a fixed token budget. In this work, we propose MS-Resampler, a multi-scope visual resampling framework for MLLMs. MS-Resampler instantiates multiple scope-specific resamplers by injecting explicit spatial scope priors into the resampling attention, enabling each branch to aggregate visual information at a particular granularity from local to global. The outputs of these scope-specific resamplers are then adaptively fused to produce the final visual representations for language modeling. Extensive experiments on ten public multimodal benchmarks show that MS-Resampler consistently improves visual understanding and multimodal reasoning over conventional single-scope resamplers, while introducing only minimal computational overhead.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：多模态大语言模型（MLLMs）通常采用“视觉编码器 + 视觉投影器 + 语言模型”的冻结骨干架构。视觉投影器负责将密集的视觉特征转换为紧凑的 token 序列，其设计直接影响计算效率和跨模态推理质量。
- **问题**：现有的重采样投影器（如 Q-Former、Matryoshka Query Transformer 等）通常使用单一、固定的聚合范围（全局交叉注意力），这种单范围假设导致：
  - 细粒度的局部证据（如小文本、微小物体、遮挡部分）被模糊；
  - 在固定 token 预算下，局部细节与全局上下文之间存在固有的权衡，无法同时有效保留两类信息。
- **核心动机**：视觉语义天然跨越多个空间范围（局部细节需要小范围，场景理解需要大范围），因此需要一种能够显式建模多范围、并在同一 token 预算下融合多层次信息的视觉投影器。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **整体框架**：MS-Resampler 由 **E 个并行的“范围重采样分支”** 和一个 **轻量级融合模块** 组成。每个分支通过注入显式的空间范围先验，在特定的粒度（从局部到全局）上聚合视觉信息，最终通过位置逐元融合生成紧凑的视觉表示。

### 核心技术细节

1. **范围重采样分支（Scoped Resampler）**
   - 在标准交叉注意力中引入 **空间注意力偏置矩阵** \( B^{(e)} \in \mathbb{R}^{M \times N} \)，控制每个查询 token 的注意力范围。
   - 每个分支 \( e \) 对应一个窗口大小 \( k^{(e)} \)，通过将视觉 token 网格划分为 \( M \) 个非重叠区域，每个查询锚定该区域中心，定义 \( k^{(e)} \times k^{(e)} \) 的偏好邻域。
   - 偏置值设为：在邻域内为 0，邻域外为 -1（软偏置，非硬掩码），通过超参数 \( \beta \) 控制强度。
   - 公式：\( O^{(e)} = \text{Softmax}\left( \frac{Q^{(e)} (K^{(e)})^\top}{\sqrt{c}} + \beta B^{(e)} \right) V^{(e)} \)

2. **多范围融合**
   - 所有分支共享相同的查询锚点，因此相同索引的 token 对应同一空间位置。
   - 对每个位置 \( i \) 和通道 \( j \)，从 E 个分支取出对应值组成向量 \( z_{i,j} \in \mathbb{R}^E \)，通过一个**共享的两层 MLP** 沿“范围（专家）维度”进行融合：\( O_{\text{final}, i,j} = w_2^\top \phi(W_1 z_{i,j} + b_1) + b_2 \)
   - 融合参数在位置间共享，仅沿范围维度操作，不混合不同位置的信息。

3. **超参数**
   - 默认配置：\( E=3 \)，窗口大小 \( \{k^{(1)}, k^{(2)}, k^{(3)}\} = \{2, 6, 24\} \)，偏置强度 \( \beta \) 通过实验调优（适中强度最佳）。
